### How do I get set up? ###

1. run dependencies
`pip install -r requirements.txt`

2. Config your .env file (only for test purpose .env has been tracked)

3. Run migrations

* 3.1 `python manage.py makemigrations`
* 3.2 `python manage.py migrate`

4. Create Superuser
`python manage.py createsuperuser`
